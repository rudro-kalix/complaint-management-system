Welcome

Welcome to the official project documentation space for the Complaint Management System for Daffodil International University, developed by Team ResolveX as part of the C Programming Complex Engineering Problem (CEP) project.

This system aims to provide a structured and efficient way for students to submit complaints, track their status, and enable administrators to manage and resolve complaints effectively. The project demonstrates practical implementation of C programming concepts including structures, file handling, functions, arrays, strings, pointers, validation, and modular programming.
