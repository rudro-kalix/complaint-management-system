# 🛠️ Complaint Management System

## Team ResolveX

Welcome to the official project repository for the **Complaint Management System for Daffodil International University**, developed by **Team ResolveX** as part of the **C Programming Complex Engineering Problem (CEP)** project.

This system provides a structured and efficient way for students to submit complaints, track complaint status, and enables administrators to manage and resolve complaints effectively. The project demonstrates practical implementation of core C programming concepts including **structures, file handling, functions, arrays, strings, pointers, validation, and modular programming**.

## 🎯 Project Objectives

* Allow students to submit complaints.
* Generate unique Complaint IDs.
* Enable complaint status tracking.
* Allow administrators to manage complaints.
* Store records using file handling.
* Improve complaint resolution workflow.
